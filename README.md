# Link Tree Project - Fundamentos Web

Este é um projeto desenvolvido com foco no domínio de **HTML Semântico** e **CSS Moderno (Flexbox)**, criado inteiramente do zero para consolidar conceitos de posicionamento e estilização.

---

##  Tecnologias Utilizadas
* **HTML5**: Estruturação semântica de elementos.
* **CSS3**: Estilização avançada e layout responsivo.
* **Flexbox**: Alinhamento e distribuição de espaço entre os componentes.

##  Aprendizados
Este repositório marca o início da minha jornada de codificação autônoma (sem auxílio de geradores de código prontos), focando em:
1. Organização de pastas e hierarquia HTML.
2. Uso de seletores CSS e efeitos de interatividade (`hover`).
3. Práticas de design responsivo.
